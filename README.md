# Bus Buddy — Admin Dashboard

## Tech Stack

| Technology   
|-------------|
 React 18    
 Tailwind CSS 
 Leaflet.js  
 Lucide React 
 Docker + Nginx 

### Running with Docker
## step 01:
git clone https://github.com/AayshaAfra/bus-buddy-dashboard.git
cd bus-buddy-dashboard

## step 02:
docker-compose up --build

## step 03:
Visit: http://localhost:3000

### Steps
 npm install
 npm start
 
##  Open in browser
## Visit: http://localhost:3000
